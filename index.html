ขั้นตอนที่ 4: การให้มูลค่าและปัจจัยปรับแก้

กำหนดมูลค่าผลลัพธ์และปัจจัยต่างๆ ในการคำนวณ

ให้มูลค่าผลลัพธ์

กรุณาเพิ่มผลลัพธ์ในขั้นตอนที่ 3 ก่อน

ปัจจัยปรับแก้ค่า

อัตราที่ใช้คำนวณมูลค่าปัจจุบัน

อัตราที่ผลกระทบลดลงในแต่ละปี (ไม่บังคับ)

สูตรการคำนวณ

ผลกระทบสุทธิ = มูลค่า × (1 - Deadweight) × (1 - Attribution) × (1 - Displacement)
มูลค่าปัจจุบัน (PV) = Σ [ผลกระทบสุทธิ / (1 + อัตราคิดลด)^ปี]
SROI Ratio = มูลค่าปัจจุบันของผลประโยชน์ / มูลค่าปัจจุบันของการลงทุน
IRR = อัตราที่ทำให้ NPV = 0

ขั้นตอนที่ 6: สร้างรายงาน

  • 1. ข้อมูลโครงการและขอบเขตการประเมิน

  • 3. แผนที่ผลกระทบ (Impact Map)
  • 4. การคำนวณและปัจจัยปรับค่า
  • 5. ผลการคำนวณ SROI
  • 6. แผนภูมิและกราฟวิเคราะห์
  • 7. สรุปและข้อเสนอแนะ
